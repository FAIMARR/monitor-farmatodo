📊 Configurar Google Sheets
Conecta la app con tu hoja de cálculo para guardar productos automáticamente
1
Crear cuenta de servicio en Google Cloud
Necesitas un archivo JSON de credenciales. Sigue estos pasos:
  1. Ve a console.cloud.google.com
  2. Crea un proyecto nuevo (ej: "Farmatodo Scraper")
  3. APIs y Servicios → Habilitar APIs → Busca y activa "Google Sheets API" y "Google Drive API"
  4. Credenciales → Crear credenciales → Cuenta de servicio → descarga el JSON
2
Crear o abrir tu Google Sheet
Crea una hoja nueva en sheets.google.com, luego compártela con el email de tu cuenta de servicio (está en el JSON, campo client_email). Dále permiso de Editor.
🤖 Configurar Análisis IA de Competidores
Usa Gemini para detectar competidores por significado, uso y propósito del producto
1
Obtener tu Gemini API Key
Ve a Google AI Studio, inicia sesión con tu cuenta Google y genera una API Key gratuita. Cópiala y pégala abajo.
💡
¿Qué detecta la IA?
La IA de Gemini analiza el nombre de cada producto y entiende para qué sirve, aunque tenga un nombre genérico o de marketing. Así puede agrupar, por ejemplo:
  • Voltarén Gel + Diclofenaco Crema → "Antiinflamatorio Tópico | 50g"
  • Panadol Tab + Dolex 500mg → "Paracetamol Oral | 500mg"
  • Agua Micelar + Garnier Agua Limpiadora → "Agua Micelar Desmaquillante"
